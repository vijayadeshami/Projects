1. What is HTML?
HTML stands for HyperText Markup Language used to create the structure of web pages. It defines elements like headings, paragraphs, images, and links in a webpage.


2. Different types of tags and elements?
There are block-level elements (e.g. <div>, <p>, <h1>) which take the full width, and inline elements (e.g. <span>, <a>, <img>) that occupy only needed width. Tags can be paired (with opening and closing) like <p></p>, or self-closing like <br>.


3. Use of iframe tag?
<iframe> is used to embed another HTML page or external content within your webpage. For example, it is used to embed YouTube videos or Google Maps.


4. Difference between <div> and <span>?
<div> is a block-level element used to group larger sections, while <span> is an inline element used to style or group small parts within a line. Use <div> for layout structures and <span> for styling specific text.


5. What is semantic HTML?
Semantic HTML uses meaningful tags like <header>, <nav>, <article> to describe the purpose of content. It improves readability, accessibility, and SEO of web pages.


6. Purpose of <meta> tags?
<meta> tags provide metadata like page description, keywords, or character set to browsers and search engines. For example, <meta charset="UTF-8"> sets character encoding and <meta name="viewport" content="width=device-width, initial-scale=1.0"> makes the page responsive.


7. Creating a hyperlink?
Use the <a> tag with the href attribute to create hyperlinks, e.g. <a href="https://example.com">Visit</a>. Common attributes include target="_blank" to open in a new tab and title for tooltip text.


8. Difference between <ol> and <ul>?
<ol> creates an ordered (numbered) list, while <ul> creates an unordered (bulleted) list. Use <ol> when the sequence matters and <ul> for general lists.


9. What is an HTML <table> element?
A table is used to display data in rows and columns. Basic structure includes <table>, <tr> for rows, <th> for headers, and <td> for data cells.


10. Purpose of <tr>, <th>, <td>?
<tr> defines a table row, <th> defines a header cell (bold and centered by default), and <td> defines a standard data cell. Together they structure the table’s rows and cells.


11. Styling tables using CSS?
You can style tables using properties like border, padding, background-color, and border-collapse. For example, table, th, td { border: 1px solid black; } adds borders to table elements.


12. HTML forms concept?
Forms collect user inputs to send to a server for processing.
a. Common elements are <input>, <textarea>, <select>, and <button> with attributes like type, name, and placeholder.
b. Form input elements include <input> for text, email, password, checkbox, radio, and submit buttons to capture different types of data.


13. What is CSS?
CSS (Cascading Style Sheets) is used to style and layout HTML elements. It complements HTML by controlling colors, fonts, spacing, and overall design.


14. Difference between inline, internal, external CSS?
Inline CSS uses the style attribute within an HTML element. Internal CSS is written inside <style> tags in the HTML head, while external CSS is in separate .css files linked to the HTML.


15. What are CSS selectors?
Selectors target HTML elements to apply styles. Class selectors use .classname, ID selectors use #id, element selectors target tags like p, attribute selectors target attributes like input[type="text"], and pseudo-class selectors like :hover style elements on specific states.


16. What is the box model?
The box model consists of content, padding, border, and margin. It defines how elements occupy space and affect layout in a webpage.


17. Difference between margin and padding?
Margin is the space outside an element’s border, creating distance from other elements. Padding is the space inside the element’s border, creating space around the content.


18. Inline vs block elements?
Inline elements do not start on a new line and only take needed width, while block elements start on a new line and take full width. You can change an element’s display using display: block; or display: inline; in CSS.


19. Different border CSS properties?
They include border-width, border-style, border-color, and shorthand border. For example, border: 1px solid black; sets width, style, and color.


20. What is Bootstrap?
Bootstrap is a popular CSS framework for building responsive and styled websites easily. It provides pre-designed components like buttons, navbars, and grids.


21. Including Bootstrap in a project?
You can include Bootstrap using its CDN link in the <head> of your HTML. For example, link its CSS via <link> tag from getbootstrap.com.


22. Containers vs container-fluid?
container provides a fixed-width layout responsive to breakpoints, while container-fluid takes full width of the viewport. Use container for centered layouts and container-fluid for full-width layouts.


23. Bootstrap grid system?
The grid system divides the page into 12 columns to create responsive layouts. It uses rows and columns with classes like .row and .col-4 to adjust layouts based on screen sizes.
